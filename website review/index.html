4. Montana

I like how Montana shrinks down when you change how big the screen is. That way it will be easily viewable from mobile devices. 5. New Hampshire

New Hampshire will shrink down a little but once it gets to a certain point it doesn’t. This could make the website hard to use from a mobile device. It does look official with the crest but overall it is still boring and I think it needs more pictures.
